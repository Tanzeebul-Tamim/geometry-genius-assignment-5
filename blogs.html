  1. 1. What is DOM and what is the purpose of DOM?

    - DOM stands for Document Object Model. A web page is represented by the DOM as a tree structure of objects, where each object stands for a document's element, attribute, or text node. Developers can dynamically change the content and structure of a web page in response to user actions or other events with the help of DOM. It offers a mechanism to access and control these objects.

  2. 2. How will you select HTML elements using DOM? Name the DOM methods.

    - Depending on the specific element we want to select and how we want it to be selected, a variety of methods can be used to select HTML elements using the DOM. Some of the most commonly used methods are:

    • a. getElementById() : The "id" attribute of a particular element is selected using this method.
    • b. getElementsByClassName() : The "class" attribute of particular elements are selected using this method.
    • c. getElementsByTagName() : Elements with a specific "tag-name" are selected using this method.
    • d. querySelector() : The first element that matches a specific"CSS selector" is selected using this method.
    • e. querySelectorAll() : All elements that matches a specific"CSS selector" are selected using this method.
  3. 3. What is event bubble?

    - In web development, event bubbling or event propagation refers to the way in which events that occur on an HTML element are propagated through the DOM tree from child elements to its parent and grand parent elements.

  4. 4. What is a callback function and why will you use it?

    - A callback function is a function that is executed by another function when a certain condition is met or an event takes place. It is passed as an argument to another function. A callback function is used to enable non-blocking code execution, allowing other pieces of code to be executed while waiting for a response or an event to happen.
